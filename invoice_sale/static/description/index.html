INVOICE/BILLS EXCEL REPORT

In this you can download invoice report based on start date, end date, invoice open or paid state and you can select whether you want to download report of customer or vendor or both and you can use "Libra Office" only.

Step 1:

When you install the module, there appears a new menu item in Invoicing --> Reporting --> PDF Reports --> Invoice Excel Report.

Step 2:

When you click on Invoice Excel Report Menu Item a wizard will open and you can see it will take by default month start date and end date and you can select whether you want to print invoice excel report on open or paid state and you can also select whether that invoice is of customer or vendor or you can also select both.


Step 3:

When you click Generate Report Button then Invoice Report Excel file will appear and you can download it by clicking on the link.

Step 4:

If you select customer in partner and when you click on the link, excel sheet is downloaded and you can see the Customer Invoice Excel Sheet. In Customer Invoice Excel Sheet you can see multiple customer invoices.



Step 5:

If you select vendor in partner and when you click on the link, excel sheet is downloaded and you can see the Vendor Bills Excel Sheet. In Vendor Bills Excel Sheet you can see multiple vendor bills.



Step 6:

If you select both in partner and when you click on the link, excel sheet is downloaded and you can see the partner excel sheet who is both a customer as well as vendor. In Excel Sheet you can see multiple customer invoices/vendor bills.



Step 7:

When you select invoice state and customer or vendor or both option for which there is no record in invoice/bills then this warning comes.

Step 8:

If you enter start date greater and end date smaller then this validation error comes.

Installation Guide

Before installing this module you have to install following packages in your system.


  • sudo pip install xlrd

  • sudo pip install xlwt

OR

  • sudo apt-get install python-xlrd

  • sudo apt-get install python-xlwt
